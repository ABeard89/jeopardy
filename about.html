What is Jeopardy?⇧

Jeopardy is a popular quiz-style game show in the United States. The game is typically played with three contestants who compete for the highest score by betting points on quiz questions. The general flow of the game is as follows:

  • One player (usually the previous winner) is chosen and given "control" of the game board. That player chooses a question by category and value (e.g. "Days of the Week for 80 points").
  • After the host has completely read the question, any player can answer the question.
    • Players to answer the question are chosen by speed, for example, the first to raise their hand or the first to ring their game buzzer (if available).
    • If they answer the question correctly, they are awarded the alotted number of points.
    • If they guess incorrectly, they lose those points (negative scores are allowed), and another player may choose to answer the question.
    • No player is forced to answer. When the question is answered correctly, or no player wishes to answer, then the correct answer is given, and the question is closed.
  • The player who correctly answers the question is given control of the board and may choose the next question.
    • If no correct answer was given on the previous question, control of the board remains with the player who chose that question.

Gameplay continues in this style until all questions have been answered.

(Note: Daily Doubles have not yet been implemented in this application.)

Application Features⇧

This application features many capabilities to enhance customization and gameplay. These features include:

  • Automatic scorekeeping with Teams
  • Ability to customize an unlimited number of Teams
  • Graphical User Interface to customize the Game Board
  • Ability to make and save an unlimited number of custom Game Boards within Data Sets
  • Automatic text size adjustment (certain areas only)
  • Ability to save data locally for future use (in the browser's cache)
  • Ability to Export Teams and Data Sets
  • Ability to Import previously exported Data
  • Fully featured "WYSIWYG"-style editor for editing questions
    • "WYSIWYG" = "What You See Is What You Get" (the editor will display it exactly as it will appear when finished, other than text color and size)
    • HTML styling support, including:
      • Bold
      • Italic
      • Underline
      • Strikethrough
      • Numbered lists
      • Bulleted lists

Recommended Browsers⇧

This application is designed to be run in a normal web browser, since every computer has one built in. Therefore, every computer, regardless of the platform, should be able to run this software. However, every browser behaves somewhat differently. Because of this, the following web browsers are recommended:

  • Google Chrome
  • Firefox
  • Safari

The following browsers are either not supported or not recommended:

  • Internet Explorer 9+ (Supported, but not recommended. May have unexpected behavior/layout)
  • Internet Explorer 8- (Not supported. Some features will not work at all.)
  • Opera (Untested, though it may work.)
